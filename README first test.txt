Setup steps

1) Download the project from github on your local machine.
2) Open project in your VS Code
3) Install node_modules package by running npm install commad via terminal in your root folder.
4) Before running the project make sure to have enviroment variables setup for Supabase in project's .env.local file. The following enviroment variables required for the Supabase:

NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=

5) After adding above mentioned enviroment variables in project's .env.local file. Run the project from your root folder via terminal by the command npm run dev.
